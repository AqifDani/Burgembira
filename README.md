# Burgembira

Burgembira is an ASP.NET Web Forms burger ordering system developed using Visual Studio 2022 and SQL Server LocalDB.

## Features

- User registration and login
- Product listing by category
- Add to cart
- Checkout
- Order history
- Delivery status
- Admin product listing page

## Technologies Used

- ASP.NET Web Forms
- C#
- SQL Server LocalDB
- HTML/CSS
- Visual Studio 2022

## How to Run

1. Clone this repository.
2. Open `Burgembira.sln` in Visual Studio 2022.
3. Restore NuGet packages.
4. Run the SQL script `tableBuild.sql` to create the database tables.
5. Start the project using IIS Express.
